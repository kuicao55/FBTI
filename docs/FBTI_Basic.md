# TasteType™ 口味人格模型 (Food MBTI)

> 版本：1.0  
> 基于四维味觉心理学模型构建，用于评估个体饮食偏好与推荐信源匹配度。

---

## 一、维度定义

TasteType 由四个独立维度构成，每个维度呈现两种对立倾向，采用单字母编码。

| 维度 | 代码 | 倾向 A | 倾向 B | 核心测量内容 |
| :--- | :--- | :--- | :--- | :--- |
| **第一维度：刺激寻求** | **S** (Spicy) / **M** (Mild) | **S型 (Spicy-Seeking)** ：追求辣椒素带来的灼烧感与痛觉快感，无辣不欢，对麻辣、辛辣有高耐受度。 | **M型 (Mild-Preferring)** ：回避疼痛型刺激，偏好温润、柔和的调味，对辣敏感或排斥。 | 对辣椒素、花椒、芥末等刺激性物质的主动寻求程度。 |
| **第二维度：口感执念** | **C** (Crunch) / **W** (Soft) | **C型 (Crunch-Seeking)** ：执着于牙齿切断食物时的脆裂感。钟爱酥、脆、硬、弹牙质地。 | **W型 (Soft-Seeking)** ：执着于舌头与上颚挤压食物的绵密感。钟爱软、糯、烂、入口即化质地。 | 口腔行为模式（咀嚼型 vs 挤压/吸吮型）。 |
| **第三维度：风味探索** | **N** (Neophilic) / **T** (Traditional) | **N型 (Neophilic)** ：对新奇食物、罕见食材、非常规搭配抱有强烈好奇心。 | **T型 (Traditional)** ：偏好熟悉的味道，认为经典菜系不可撼动，对陌生食物持谨慎态度。 | 食物新奇恐惧症（Food Neophobia）量表倾向。 |
| **第四维度：鲜味敏感** | **U** (Umami) / **P** (Plain) | **U型 (Umami-Driven)** ：对谷氨酸盐（鲜味）高度敏感。依赖高汤、发酵酱料、奶酪、肉类带来的“厚味”。 | **P型 (Plain-Sensitive)** ：对鲜味添加剂（味精）或浓烈发酵味敏感甚至反感，偏好清鲜、本味、蔬食感。 | 对T1R1/T1R3基因相关的鲜味感知偏好。 |

## 二、16 种口味人格全列表

### 🟤 S-C-N-U 型：烈焰暴君 (The Pyroclastic Tyrant)
- **味觉签名**：S(嗜辣) + C(嗜脆) + N(猎奇) + U(嗜鲜)
- **行为画像**：此类型是人类味觉边疆的拓荒者。追求极致的复合型感官轰炸——必须辣到流汗、脆到出声、鲜到掉眉毛。他们热衷于寻找“变态辣脆皮肥肠”或“烟熏辣椒巧克力脆片”。推荐信源可信度极高，但普通人跟上需要勇气。
- **避雷区**：清蒸、白灼、汤羹类食物。W型人的软烂推荐对他们来说如同“婴儿辅食”。

### 🟠 S-C-N-P 型：硬核极客 (The Hardcore Geek)
- **味觉签名**：S(嗜辣) + C(嗜脆) + N(猎奇) + P(厌鲜)
- **行为画像**：他们追求纯粹的化学刺激与物理脆度，但对“鲜”非常苛刻，讨厌味精感和过度的发酵味。他们可能爱吃“死神辣椒薯片”但极度厌恶鸡精。他们推荐的辣馆子通常“干净利落、辣得纯粹”。
- **避雷区**：浓汤宝、松露油、老火靓汤。

### 🟡 S-C-T-U 型：江湖老饕 (The Old-School Gourmand)
- **味觉签名**：S(嗜辣) + C(嗜脆) + T(经典) + U(嗜鲜)
- **行为画像**：地方菜系的坚定捍卫者。只认可“正宗”的辣脆结合，比如重庆火锅里的毛肚黄喉、湖南小炒里的脆骨。拒绝为了猎奇而改良的融合菜。他们对特定菜系的推荐具有权威性。
- **避雷区**：任何“新派”或“分子料理”版本的家乡菜。

### 🟢 S-C-T-P 型：盐系刺客 (The Salty Assassin)
- **味觉签名**：S(嗜辣) + C(嗜脆) + T(经典) + P(厌鲜)
- **行为画像**：纯粹的重口味盐分爱好者。比起鲜，更爱直接的咸辣。他们热爱的是夜市里的“疯狂烤翅”、校门口的“油炸淀粉肠”。他们对环境不挑，只看“味道够不够劲儿”。
- **避雷区**：精致日料、需要细品的本味食材。

### 🔵 S-W-N-U 型：熔岩核心 (The Lava Core)
- **味觉签名**：S(嗜辣) + W(嗜软) + N(猎奇) + U(嗜鲜)
- **行为画像**：这种类型渴望“内在的燃烧”。他们最爱的食物是：炖得软烂入味的辣卤鸡爪、入口即化的水煮脑花、吸饱汤汁的臭豆腐。追求辣与鲜在绵密口感中的缓慢释放。
- **避雷区**：沙拉、干煎、任何需要用力咀嚼的食物。

### 🟣 S-W-N-P 型：味蕾实验员 (The Palate Tester)
- **味觉签名**：S(嗜辣) + W(嗜软) + N(猎奇) + P(厌鲜)
- **行为画像**：他们可能为了尝试“辣椒冰淇淋”而去排队，或者喜欢吃“软糯的韩式辣年糕”。他们喜欢新奇辣味带来的趣味性，但不希望菜里有“药膳味”或“高汤味”。
- **避雷区**：药材炖汤、浓郁菌菇锅底。

### 🔴 S-W-T-U 型：红汤居士 (The Red Broth Hermit)
- **味觉签名**：S(嗜辣) + W(嗜软) + T(经典) + U(嗜鲜)
- **行为画像**：四川火锅里的“耙牛肉”、湖南的“口味虾”汤汁拌饭是他们的归宿。他们是那种会因为一道“老妈蹄花汤不够软烂”而给差评的人。他们对“软”和“辣”的比例有严格的审美标准。
- **避雷区**：西式牛排、生硬的沙拉菜。

### 🟤 S-W-T-P 型：刺激渴求者 (The Stimulus Craver)
- **味觉签名**：S(嗜辣) + W(嗜软) + T(经典) + P(厌鲜)
- **行为画像**：他们吃辣只是为了爽，为了多巴胺。最喜欢的食物是：软糯的麻辣烫宽粉、吸满辣椒油的凉皮。对于食材本身是否高级、是否有鲜味不感兴趣。
- **避雷区**：价格昂贵但辣度不足的餐厅。

### 🟠 M-C-N-U 型：风味猎人 (The Flavor Hunter)
- **味觉签名**：M(拒辣) + C(嗜脆) + N(猎奇) + U(嗜鲜)
- **行为画像**：**这是你提到的“德国烤土豆”人格。** 完全不吃辣，但追求极高风味浓度。他们热爱焦脆的芝士脆皮、美拉德反应极致的煎烤、复杂的香料组合（不含辣）。推荐非辣菜系的异国料理极准。
- **避雷区**：白水煮菜、寡淡的减肥餐。

### 🟡 M-C-N-P 型：清爽极客 (The Fresh Geek)
- **味觉签名**：M(拒辣) + C(嗜脆) + N(猎奇) + P(厌鲜)
- **行为画像**：他们喜欢口感上的趣味，但讨厌味道上的负担。他们可能会迷恋“海盐焦糖脆片”或“柚子味气泡炸鸡皮”。他们推荐的餐厅通常“有趣且不油腻”。
- **避雷区**：红烧、卤煮等重色重味的菜。

### 🟢 M-C-T-U 型：经典脆党 (The Classic Cruncher)
- **味觉签名**：M(拒辣) + C(嗜脆) + T(经典) + U(嗜鲜)
- **行为画像**：粤菜烧腊的忠实拥趸。烧鹅皮、乳猪皮、炸子鸡皮是他们的灵魂伴侣。他们重视食材本味的鲜与烹饪技法的脆。他们推荐的粤菜/淮扬菜馆子可信度满分。
- **避雷区**：炖得太烂的肉、没有口感对比的菜。

### 🔵 M-C-T-P 型：纯粹口感派 (The Texture Purist)
- **味觉签名**：M(拒辣) + C(嗜脆) + T(经典) + P(厌鲜)
- **行为画像**：极度挑食的“小孩口味”成人版。只爱吃薯条、炸猪排、脆皮面包。对复杂的调味（哪怕是黑胡椒）都可能排斥。他们的推荐仅限于“炸物榜”。
- **避雷区**：除了油炸和烘烤脆壳的一切。

### 🟣 M-W-N-U 型：醇厚鉴赏家 (The Mellow Connoisseur)
- **味觉签名**：M(拒辣) + W(嗜软) + N(猎奇) + U(嗜鲜)
- **行为画像**：**这是你提到的“漏奶华/巧克力”人格。** 他们用“软”和“浓”来衡量幸福。热爱提拉米苏、勃艮第红酒炖牛肉、溏心蛋、葱烧海参。他们是甜品店和西餐厅的优质推荐信源。
- **避雷区**：筋道的面条、法棍、麻辣重口。

### 🔴 M-W-N-P 型：轻盈梦想家 (The Ethereal Dreamer)
- **味觉签名**：M(拒辣) + W(嗜软) + N(猎奇) + P(厌鲜)
- **行为画像**：喜欢尝试各种口味的布丁、慕斯、云朵蛋糕。追求的是“空气感”和“温柔的甜/酸”。他们对食物的视觉美感要求往往高于味道浓度。
- **避雷区**：油腻、扎实、管饱的主食。

### 🟤 M-W-T-U 型：怀旧治愈家 (The Nostalgic Healer)
- **味觉签名**：M(拒辣) + W(嗜软) + T(经典) + U(嗜鲜)
- **行为画像**：这是最典型的“外婆的味道”人格。番茄炒蛋拌饭、红烧肉汁拌饭、烂糊面、鸡蛋羹。他们通过软烂的食物获取安全感。他们推荐的“家常菜馆”往往环境一般但味道窝心。
- **避雷区**：生冷、有嚼劲、创新的融合菜。

### ⚪ M-W-T-P 型：极简隐士 (The Minimalist Hermit)
- **味觉签名**：M(拒辣) + W(嗜软) + T(经典) + P(厌鲜)
- **行为画像**：饮食世界的“性冷淡风”。可能最喜欢白粥配腐乳（只吃腐乳的咸味不吃豆子）、白吐司、土豆泥（只放盐）。他们不是挑剔，只是味觉阈值极低且对多数风味物质敏感。
- **避雷区**：几乎所有的社交型餐饮。

## 三、算法应用：推荐信源匹配度公式

在软件后台，将用户 A 和推荐人 B 的类型进行比对：

> **匹配度评分规则：**
> - 四个字母完全相同：**100% (灵魂食伴)**
> - 三个字母相同：**75% (高度可信)**
> - 两个字母相同：**50% (存在雷区需警惕)**
> - 一个字母相同：**25% (仅供参考，大概率踩雷)**
> - 零个字母相同：**0% (建议反向操作，他说好吃的你别去)**

**特殊权重提示：**
- 如果 **第一维度(S/M)** 不一致，关于“辣度”的描述需要自动反向翻译（他说微辣=你说爆辣）。
- 如果 **第二维度(C/W)** 不一致，关于“口感”的形容词（脆/糯）需完全忽略。
