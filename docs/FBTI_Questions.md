# TasteType™ 口味人格测试题库与计分逻辑

> 本测试包含 16 道情景选择题，预计用时 3-5 分钟。  
> 所有题目无对错之分，请根据第一直觉反应作答。  
> 最终输出 4 位字母组成的口味人格类型。

---

## 一、测试引导语

请在以下情境中，选出**最接近你真实反应**的选项。如果多个选项都符合，请选择你**最无法忍受被剥夺**的那一项。

---

## 二、测试题目

### 【第一维度：刺激寻求 (S/M) —— 4 题】

**Q1. 一碗刚出锅的红油抄手端上来，你的第一注意力落在哪里？**  
A. 那层亮红色的辣椒油和漂浮的芝麻，已经自动分泌口水了  
B. 先把辣椒拨开一点，看看里面的肉馅和汤底清不清  
C. 我更关心抄手皮是薄是厚、是软是脆  
D. 我会先闻味道，判断这是单纯的辣还是有花椒、香料等复合香气  

**Q2. 吃火锅调蘸料时，你的碗里通常是什么画风？**  
A. 小米辣、蒜泥、香油、蚝油、香菜葱花都要堆满  
B. 酱油底，最多加一点点辣椒圈提味  
C. 芝麻酱+腐乳+韭菜花，或沙茶酱+蛋黄，不太放辣椒  
D. 看锅底决定，牛油锅就蘸干碟，清汤锅就蘸海鲜汁  

**Q3. 当你看到菜单上有“魔鬼辣挑战成功免单”的活动时，你的想法是？**  
A. 让我来，这是我的主场  
B. 除非朋友起哄，否则我不想折磨自己  
C. 我觉得这会影响品尝食物本身的味道，不参加  
D. 先研究一下辣椒品种，如果是纯工业辣精就不玩，天然辣椒可以试试  

**Q4. 吃完一顿重辣餐后，你的身体通常是什么反应？**  
A. 嘴巴肿了但心里爽了，第二天继续  
B. 肠胃可能会抗议，我尽量控制频率  
C. 我会大量喝水，并且下一顿一定要吃清淡的缓一缓  
D. 看辣椒品质，好辣椒只辣嘴不烧心，坏辣椒会让我难受很久  

> **维度指向**：选 A 得 S 分，选 B/C 得 M 分，选 D 需结合其他题判断（倾向 S 但具备克制意识）。

### 【第二维度：口感执念 (C/W) —— 4 题】

A. 牙齿咬下去发出“咔嚓”声，脆皮碎屑掉下来  
B. 里面的肉汁瞬间烫到舌头，肉质嫩滑到不用怎么嚼  
C. 我喜欢先吃皮再吃肉，皮必须脆，肉可以稍干  
D. 我其实不爱油炸，我更爱红烧或炖煮的口感  

**Q6. 如果一碗红烧肉让你评价，你觉得最不能忍受的缺点是？**  
A. 瘦肉部分太柴，塞牙  
B. 肥肉部分没有炖烂，咬起来像肥油块  
C. 皮上的毛没去干净  
D. 汤汁不够浓稠，挂不住肉  

**Q7. 你对“软糯”类食物（如土豆泥、年糕、烩饭）的态度是？**  
A. 可以吃，但不能连续吃，否则觉得嘴巴没劲  
B. 这是我的舒适区，我可以天天吃软软的东西  
C. 看搭配，如果是软糯主食配脆的小菜，我就很爱  
D. 我更关心调味，软或硬没那么重要  

**Q8. 吃拉面时，你对“面条硬度”的选择通常是？**  
A. 必须点“超硬”或“硬”，喜欢吃有嚼劲的  
B. 点“普通”或“偏软”，好嚼好消化  
C. 第一次去会点普通，之后再根据汤头调整  
D. 我不吃拉面，或者我不关心这个细节  

> **维度指向**：连续选 A/C（偏向脆硬）得 C 分，连续选 B（偏向软烂）得 W 分。若选 D 过多，需根据 Q5/Q6 权重题强制归边。

### 【第三维度：风味探索 (N/T) —— 4 题】

**Q10. 朋友向你安利一家“用巧克力蘸辣椒面吃”的店，你的第一反应是？**  
A. 听起来很暗黑，但我必须去验证一下

**Q11. 你对“正宗”这个词的态度是？**  
A. 正宗很重要，那是文化和风味的根基  
B. 好吃就行，我不在乎正不正宗，改良得好我也爱  
C. 有些菜必须正宗（比如我妈做的），有些菜可以乱来

**Q12. 在自助餐厅，你的取餐策略通常是？**

**Q15. 吃日料刺身时，你会怎么蘸酱油？**  
A. 酱油里一定会加现磨山葵，还要蘸很多下  
B. 轻轻蘸一点酱油，甚至有的鱼不蘸直接吃

## 三、计分规则

每个维度独立计分，满分为 4 分（每题倾向选项得 1 分，若某维度 4 题中有 3 题指向同一倾向，则该倾向得分 +1）。

| 维度 | 计分项 | 判定阈值 |
| :--- | :--- | :--- |
| **第一维度** | S 得分 = Q1~Q4 中选 A 的次数 + (选 D 的次数 × 0.5) | S ≥ 2.5 则记为 **S**，否则 **M** |
| **第二维度** | C 得分 = Q5~Q8 中选 A/C 的次数（优先 Q5/Q6 权重） | C ≥ 2.5 则记为 **C**，否则 **W** |
| **第三维度** | N 得分 = Q9~Q12 中选 A/B 的倾向（A=+1N，B=+1T，C/D=+0.5） | N ≥ 2.5 则记为 **N**，否则 **T** |
| **第四维度** | U 得分 = Q13~Q16 中选 A 的次数（选 B 计为 P 倾向） | U ≥ 2.5 则记为 **U**，否则 **P** |

> **注意**：若某维度出现 2:2 平局，则引入**加试题逻辑**或默认取**第一题倾向**。

## 四、类型判定伪代码

以下为工程师可直接参考的判定逻辑（Python 风格伪代码）：

```python
def calculate_taste_type(answers):
    # answers: list of 16 chars ['A','B','C','D', ...]
    
    # 1. 计算 S 倾向
    s_score = 0
    for i in [0,1,2,3]:  # Q1-Q4 索引
        if answers[i] == 'A':
            s_score += 1
        elif answers[i] == 'D':
            s_score += 0.5
    dim1 = 'S' if s_score >= 2.5 else 'M'
    
    # 2. 计算 C 倾向 (Q5-Q8)
    c_score = 0
    for i in [4,5,6,7]:
        if answers[i] in ['A', 'C']:
            c_score += 1
        # Q5权重题：选A额外加0.5，选B扣0.5
        if i == 4 and answers[i] == 'A':
            c_score += 0.5
        if i == 4 and answers[i] == 'B':
            c_score -= 0.5
    dim2 = 'C' if c_score >= 2.5 else 'W'
    
    # 3. 计算 N 倾向 (Q9-Q12)
    n_score = 0
    for i in [8,9,10,11]:
        if answers[i] == 'A':
            n_score += 1
        elif answers[i] == 'B':
            n_score -= 1
        # C/D 不改变分数，视为中性
    dim3 = 'N' if n_score >= 1 else 'T'  # 阈值调为1因存在负分可能
    
    # 4. 计算 U 倾向 (Q13-Q16)
    u_score = 0
    for i in [12,13,14,15]:
        if answers[i] == 'A':
            u_score += 1
        elif answers[i] == 'B':
            u_score -= 1
    dim4 = 'U' if u_score >= 1 else 'P'
    
    # 组合返回
    taste_type = dim1 + dim2 + dim3 + dim4
    return taste_type
```
## 五、边界情况处理建议

1. 全部选中立项（社会称许作答）：若 16 题全部选 A 或全部选 B，系统应判定为“无效作答”，提示用户根据真实感受重新测试。

2. 中间派过多：若四个维度中有三个出现平局，可弹出一个额外的“二选一”迫选题以打破平局。

3. 跳过题目：不允许跳过，必须全部作答。
